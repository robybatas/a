Pengenalan Hacktivate dan Tugas Pertama

Phase 0, week 1, day 1

Hari pertama dimulai dengan instruktur Hacktivate memberikan breifing kepada seluruh peserta bootcamp. Briefing berupa pengenalan hacktivate diasampaikan dengan cukup baik oleh para instruktur. Peserta bootcamp tampak antusias, beberapa bahkan mengajukan pertanyaan.

Sesi briefing ini merupakan sesi yang sangat penting. Setelah mengikuti sesi ini, saya secara pribadi merasa lebih kenal dan dekat dengan Hacktivate.

Pada sesi ini peserta bootcamp diperkenalkan dengan berbagai hal mengenai Hacktivate, mulai dari bagaimana program pembelajaran di Hacktivate umumnya berjalan hingga berbagai varian metode pembayaran yang dapat dipilih oleh peserta bootcamp. Meskipun saya sudah pernah mengikuti sesi serupa pada saat mendaftar program ISA, saya merasa sangat terbantu dengan dibarikannya sesi pengenalan di hari pertama mengikuti bootcamp.

Student Card

Setelah mengenal Hacktivate, peserta bootcamp diarahkan untuk melakukan psikotes, lalu kemudian dibimbing untuk mengisi student card. Kegunaan dari card tersebut adalah untuk memudahkan Hacktivate menilai "peserta bootcamp saat ini", mulai dari lokasi tempat tinggal hingga alasan dan tujuan mengikuti bootcamp menjadi konten isian wajib bagi seluruh peserta. informasi-informasi tersebut sebenarnya diperlukan untuk memudahkan Hacktivate membantu peserta dalam menjalankan proses bootcamp. Fasilitas Hacktivate semisal engineering emphaty sangat membutuhkan infomasi yang bersumber dari hasil psikotes dan data pada student card.

Adanya sistem penilaian dan pemantauan melalui student card, membuat saya secara pribadi merasa aman mengikuti bootcamp. Student crad bagi saya merupakan realisasi dari komitmen Hacktivate kepada peserta bootcamp untuk membatu proses pembelajaran bahkan pada aspek lain yang menunjang proses akademik peserta.

Git dan Github

Selanjutnya peserta bootcamp diperkenalkan dengan beberapa command pada command line. Setelah familiar dengan beberapa command dan penggunaan command line, peserta bootcamp diperkenalkan dan diajarkan penggunaan sederhana dari Git dan Github.

Hari pertama ditutup dengan tugas sederhana yang berhubungan dengan Git dan Github tentunya. Penasaran tugasnya apa? Ya gabung Hacktivate lah! Tapi tenang, belum ada ngoding-ngoding atau belajar logic yang berat-berat ya guys ya hehe
